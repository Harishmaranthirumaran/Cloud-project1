# Backup and Restore Governance Platform

## Keynote

This project shows how to operationalize recovery on AWS. It combines backup plans, vaults, retention, and a protected S3 workload so the portfolio includes a genuine recovery and resilience story.

## Best for

- Reliability engineer
- Senior cloud engineer
- DevOps engineer with recovery ownership

## Core AWS services

- AWS Backup
- Amazon S3
- AWS KMS
- IAM
- Terraform

## What it proves

- Backup policy design
- Restore-oriented thinking
- Retention and protection controls
- Operational recovery readiness
