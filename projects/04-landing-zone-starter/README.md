## Build prompt

> Create a portfolio-grade AWS landing zone starter using Terraform. Model a small multi-account setup with management, security, shared-services, dev, and prod accounts. Include centralized logging, CloudTrail, Config, GuardDuty, baseline KMS encryption, tagging standards, budget-conscious defaults, and architecture documentation. Keep it realistic but feasible for a portfolio project rather than a full enterprise implementation.
