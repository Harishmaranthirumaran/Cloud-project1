AWS Static Website Platform

Fast delivery, private origin, clean deploys.

A production-style static site stack with CloudFront, S3, Route 53, ACM, and GitHub Actions.

Explore stack Deployment flow
Origin

Private S3 bucket

Stores static assets behind CloudFront Origin Access Control so the bucket is not public.

Delivery

CloudFront CDN

Adds TLS, caching, and edge delivery for a better first-byte experience.

DNS

Route 53 optional

Custom domain support is available when you are ready to attach a hosted zone.

Deploy

GitHub Actions

OIDC-based deploys keep credentials out of the repo and invalidate the CDN after upload.

Deployment flow

  1. GitHub Actions authenticates to AWS with OIDC.
  2. The workflow syncs `app/` contents to S3.
  3. CloudFront invalidation refreshes the live site.
  4. Users receive cached content at the edge.
